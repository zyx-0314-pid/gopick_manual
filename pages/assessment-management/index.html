Assessment Randomization and Timer

Configure question randomization and timer settings for any assessment.

1
Open "View Assessments"
Locate it in the side panel of the admin portal.
2
Select an assessment
Choose the assessment from the gallery view.
3
Open configuration
Click the configuration icon — the 3rd button in the top-left button group.
4
Select "Page Setup"
From the dropdown, select Page Setup where you can configure Timer and Randomization settings.

Upload Images

Super Admin Only

Upload or update assessment images. Restricted to ASD and IT Super Admin roles.

1
Open "View Assessments"
Locate it in the side panel of the admin portal.
2
Select an assessment
Choose the assessment from the gallery view.
3
Open settings
Click the settings icon — the 4th button in the top-left button group.
4
Select "Update"
From the dropdown, select Update.
5
Upload under Assessment Development
Below the ASSESSMENT DEVELOPMENT heading, click the upload icon to add or update images.
