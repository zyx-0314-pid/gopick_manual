Configuration Steps

How to view and update role permissions in the Admin portal.

1
Open Roles/Permissions
Open the Roles/Permissions module in the Admin console.
2
Search and select a role
Use Search Roles to find the role you want to change.
3
Open configuration
Select the role row, then click Edit or Preview.
4
Locate permissions
In the Edit view, locate the Permissions list or table.
5
Update permission
Find the entry by name/module and toggle access. For conditional permissions, update the condition.
6
Save and verify
Save changes and use Preview or impersonation to confirm the role behavior.
