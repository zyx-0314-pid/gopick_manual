Role Capabilities

Configuration Steps

1
Open "Roles/Permissions"
Open the Roles/Permissions module in the Admin console.
2
Search and select a role
Use Search Roles to find the role you want to change (e.g., "Administrator", "Distributor").
3
Open configuration
Select the role row, then click the Edit or Preview button.
4
Locate permissions
In the Edit view locate the Permissions list or table.
5
Find and update permission
Find the exact permission entry by name and module, then toggle or update it as required.
6
Conditional permissions
If the permission is marked Conditional, update its condition or configuration so it becomes available; otherwise toggle the direct access flag (grant/revoke).
7
Save and verify
Save your changes and use the Preview or impersonation to confirm the role's behavior.
