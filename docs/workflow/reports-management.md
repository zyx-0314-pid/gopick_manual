# Usage Module (Reports)

## Search Usage
Search and review assessment usage records, including candidate, company, assessment, schedule date, scheduler, and assessment status.

> Rules:
> 2 Types of assessment displayed in 2 ways under Assessment Column\
> - **Group Assessment/Test Battery**: Displayed to have `Test Battery` name above the sub-assessment name
> - **Single Assessment**: Displayed as simple as the assessment name 

### Exporting Reports
Download available assessment reports for selected candidates and assessments.

1. Select `Report` > `Search Usage`
2. Select Assessments
3. In `Bulk Action` drop down select `Download Report`

### Sending Reports
Send assessment reports to selected recipients for candidates with available generated reports.

1. Select `Report` > `Search Usage`
2. Select Assessments
3. In `Bulk Action` drop down select `Send Report`

### Export Assessment Data
Download assessment reports for selected assessment with its date range

1. Select `Report` > `Search Usage`
2. Select `Export Assessment Data`
3. Select `Account`, `Assessment` in the account and `Date Range`
4. Export Data
