## Assessment Configurations
### Assessment Randomization and Timer
1. select `View Assessments` side panel 
2. select assessment from the gallery
3. select configuration icon (quadrant 2 - top left - 3 icon from group of button left to right)
4. select `Page Setup` from drop down
5. select what part of the exam will be randomized and has timer and specified time limit

from there you can set `Timer` and also `Randomization`

### Upload Images for Assessments (Super Admin Exclusive - ASD & IT) 
1. select `View Assessments` side panel 
2. select assessment from the gallery
3. select settings icon (quadrant 2 - top left - 4 icon from group of button left to right)
4. select `Update` from drop down
5. below `ASSESSMENT DEVELOPMENT` there is upload icon. There you can update it

### Updating Instructions on Assessments 
1. select `View Assessments` side panel 
2. select assessment from the gallery
3. select configuration icon (quadrant 2 - top left - 3 icon from group of button left to right)
4. select `Set Questions` from drop down
5. under the `Question` you can fill it with configurations you want
