# Meters Module

## Meter Requests
List of requests of lower in hierarchy and able to `reject` or `approve` request.

1. Meters Module > Meter Request
2. Select which request to `approve` or `reject`

## Order Meters
Request list of meters from higher ups in hierarchy and list of request history

1. Input value request
2. Click `Send Meter Order`

## Transfer Meters
Transfer Meter balance to child accounts.

1. Meters Module > Meter Request
2. Select `Source Account`
    - `Available Meters` will be loaded here
3. Enter `Meter Value` to transfer
4. Select `Destination Account`
5. Click `Transfer Meter`

## Meter Records
Table list of all children account that can view their `meter` or `meter log`

### View Meter
View Details of meters of the account

### View Meter Log
View all logs of meters
